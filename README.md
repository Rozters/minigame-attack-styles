# Minigame Attack Styles
When using Attack Styles Plugin users are able to hide warned attack styles for warned styles.
This plugin extends Attack Styles Plugin to ignore warned styles when inside a minigame with exp disabled.
User selects the minigame this addon will work on.
- Clan Wars
- Last Man Standing
- Bounty Hunter
